# ngx-admin-charts

Angular 19 Charts Demo - migrated from ngx-admin

This project migrates the Charts section from [ngx-admin](https://github.com/akveo/ngx-admin) to Angular 19, replacing Nebular with Bootstrap and updating all chart libraries to their latest versions.

## Chart Libraries
- **ECharts** (echarts v5 + ngx-echarts v18)
- **Chart.js** (chart.js v4 + ng2-charts v6)
- **D3/ngx-charts** (@swimlane/ngx-charts v20)

## Development
```bash
npm install
ng serve
```

Navigate to http://localhost:4200/pages/charts/echarts
